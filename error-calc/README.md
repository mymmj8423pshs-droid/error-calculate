# 실험 오차 계산기

실험값·이론값을 입력하면 오차율·표준편차·신뢰구간·이상치·t-검정과 기준점 대비 효율 분석, 보고서용 문장까지 자동으로 생성하는 웹 도구.

## 기능

- **실험값 ↔ 이론값**: 평균, 표본표준편차, 표준오차, 절대/상대오차, 95% 신뢰구간, 이론값-신뢰구간 일치 판정
- **실험값 ↔ 실험값**: Welch t-검정 (t, df, p값, Cohen's d), 평균 차이(%)
- **기준점 대비 효율**: 기준점(100%) 대비 잔류율·성능 향상률, 효율 순위 자동 정렬 (낮을수록 좋음 / 높을수록 좋음 방향 선택)
- **이상치 검출**: IQR 기준 자동 표시
- **그래프**: 오차막대 막대그래프, 산점도, 히스토그램, 효율 순위 시각화
- **보고서 문장 자동 생성**: 한국어 조사 자동 처리, 전체 복사

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포 (Vercel)

1. 이 폴더를 GitHub 레포로 푸시
2. Vercel에서 해당 레포를 Import
3. Framework Preset이 자동으로 **Vite**로 잡힘 (Build: `npm run build`, Output: `dist`)
4. Deploy

API 호출이 없는 순수 프론트엔드라 환경변수 설정 불필요.

## 기술 스택
